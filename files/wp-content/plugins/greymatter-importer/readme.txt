== Description ==

Import users, posts, and comments from a Greymatter blog.

== Installation ==

1. Upload the `greymatter-importer` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to the Tools -> Import screen, Click on GreyMatter
